# soft-75

A simple daily habit tracker for a soft version of the 75 Hard program.

## Quickstart

```bash
npm install
npm run dev
```

## Usage

1. Create a Supabase project and run the SQL in `supabase/schema.sql`.
2. Copy `.env.example` to `.env.local` and set your Supabase URL and anon key.
3. Start the app with `npm run dev`.
4. Pick a user and track daily completion for exercise, mindfulness, water, diet, and journal.

## Tech stack

- TypeScript
- Next.js
- React
- Tailwind CSS
- Supabase
- SWR
